## Code - Starter  
Bolierplate for create new projects base on:  
- NodeJS
- Vuejs (https://vuejs.org)
- Hapi (hapijs.com)
- Webpack (https://webpack.js.org/)
- Scss 
- Bulma (http://bulma.io/)

### Run

```
npm install
```

```
npm run build
```  

On another terminal window:
```
npm run start
```
